“Barrera” es una de las habilidades de mejora de estado más eficaces de Natalia, así que equípale una cámara cobalto para aumentar la duración del hechizo. Añade una cámara prado a “Calvario” para añadirle un efecto sanador, para hacer de esta habilidad mediocre un hechizo un poco más útil. Finalmente, equipa una cámara prado a su “Láser aéreo” para que robe objetos, aunque es un poco difícil de utilizar.
